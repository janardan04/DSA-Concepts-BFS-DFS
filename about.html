Depth First Search (DFS)

DFS is an algorithm for traversing or searching tree or graph data structures. Starting from the root node, it explores as far as possible along each branch before backtracking.

Algorithm in C:
void DFS(int graph[][V], int visited[], int node) {
    visited[node] = 1;
    printf("%d ", node);
    
    for (int i = 0; i < V; i++) {
        if (graph[node][i] == 1 && !visited[i]) {
            DFS(graph, visited, i);
        }
    }
}
Explanation:

DFS uses a stack data structure for backtracking and explores as deep as possible in each branch before returning to check other nodes.

History:

DFS was introduced by the computer scientist Robert Tarjan in the 1970s as part of his work in graph theory.

Breadth First Search (BFS)

BFS is an algorithm for traversing or searching tree or graph data structures. It starts at the root node and explores all neighboring nodes at the present depth before moving on to nodes at the next depth level.

Algorithm in C:
void BFS(int graph[][V], int start) {
    int visited[V] = {0};
    int queue[V], front = -1, rear = -1;
    
    visited[start] = 1;
    queue[++rear] = start;
    
    while (front != rear) {
        int node = queue[++front];
        printf("%d ", node);
        
        for (int i = 0; i < V; i++) {
            if (graph[node][i] == 1 && !visited[i]) {
                visited[i] = 1;
                queue[++rear] = i;
            }
        }
    }
}
Explanation:

BFS uses a queue data structure to explore nodes level by level, ensuring all neighbors are explored before moving deeper into the graph.

History:

BFS was first described by the mathematician and computer scientist Edward F. Moore in 1959 as a part of his work in search algorithms.
