What is a Graph in DSA?

A graph is a data structure that consists of a set of nodes (vertices) and edges connecting them. It can represent relationships or networks like social networks, roads, and web pages.

Graph Traversal

Graph traversal involves visiting all vertices or nodes in a graph. The two main methods are Depth First Search (DFS) and Breadth First Search (BFS).

DFS and BFS Traversal

DFS: Uses a stack to explore as far as possible before backtracking.

BFS: Uses a queue to explore all neighbors before moving to the next level.
